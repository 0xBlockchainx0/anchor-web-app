# Anchor Icon Library

These icons are compatible with material-ui's icon.

```tsx
import { IconButton } from '@material-ui/core';
import { Wallet } from '@anchor-protocol/icons';

function App() {
  return (
    <IconButton>
      <Wallet />
    </IconButton>
  );
}
```
